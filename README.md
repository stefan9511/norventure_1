## Getting Started

### Installation

```bash
npm install
```

### Development

Start the development server:

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build

Build for production:

```bash
npm run build
```

### Preview

Preview the production build:

```bash
npm run preview
```

## Deployment

This site is configured for deployment to GitHub Pages. See [DEPLOYMENT.md](./DEPLOYMENT.md) for detailed instructions.

Quick deployment steps:
1. Create a GitHub repository
2. Push your code
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://YOUR_USERNAME.github.io/norventure/`
